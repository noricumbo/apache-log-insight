# Roadmap

This project aims to stay lightweight and practical. The roadmap focuses on features that improve visibility and usability without turning the project into a full SIEM.

## Planned ideas

- Click IP to show an attack breakdown
- Per-IP timeline
- Export incident report as Markdown
- Configurable category rules
- Nginx log support
- GeoIP support
- Docker deployment
- Better mobile layout
- Dark and light theme toggle
- Dashboard auth helper examples

## Notes

- Some features may stay optional to preserve the simple deployment model.
- Changes should continue to favor plain Python, static output, and easy self-hosting.
